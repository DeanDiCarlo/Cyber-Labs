Hands-on cybersecurity labs and writeups focused on practical, ethical skill-building:
- Network reconnaissance and enumeration
- Web application testing (OWASP basics)
- Packet analysis
- Linux security + log triage
- Intro incident response workflows

> **Ethics / Scope:** All work in this repository is performed in legal lab environments (local VMs, Docker labs, or authorized training platforms). No testing is conducted against real-world systems without explicit permission.

## Repo Structure
- `writeups/` — case-study style lab writeups with evidence and remediation notes
- `resources/` — setup notes, references, and personal cheatsheets
- `scripts/` — helper notes/scripts (sanitization, workflow helpers)
